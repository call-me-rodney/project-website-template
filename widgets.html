BLIND AID (VERSION 4)

Hardware Connection Guide for Blind Aid Version 4

The following table outlines the connections for the Blind Aid’s components to the Arduino Uno and power system.

Component Connection Details
Li-Po Battery (2000mAh) B+ → TP4056 B+, B- → TP4056 B-
TP4056 Charging Module Micro-USB/USB-C → 5V charger, OUT+ → P-MOSFET Source, OUT- → Boost Converter Vin-, Arduino GND
P-MOSFET (IRF9540N) Source → TP4056 OUT+, Drain → Boost Converter Vin+, Gate → Arduino D3 via 10kΩ resistor
Push Button Leg 1 → Arduino GND, Leg 2 → Arduino D3
MT3608 Boost Converter Vin+ → P-MOSFET Drain, Vin- → TP4056 OUT-, Arduino GND, Vout+ → Arduino VIN (5–7V), Vout- → Arduino GND
HC-SR04 VCC → Arduino 5V, GND → Common ground, TRIG → D6, ECHO → D7
VL53L0X VCC → Arduino 5V, GND → Common ground, SDA → A4, SCL → A5
MPU6050 VCC → Arduino 3.3V, GND → Common ground, SDA → A4, SCL → A5
PWM Vibration Motor VCC → Arduino 5V, GND → Common ground, PWM → D9
SIM800L EVB 5V → Arduino 5V, GND → Arduino GND, TX → D10, RX → D11 (SoftwareSerial)
